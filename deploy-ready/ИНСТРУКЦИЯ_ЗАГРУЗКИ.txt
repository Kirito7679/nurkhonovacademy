🚀 ШАГИ ЗАГРУЗКИ:
───────────────────────────────────────────────────────────────

СПОСОБ 1: Через cPanel File Manager
───────────────────────────────────────────────────────────────

1. Войдите в cPanel → File Manager
2. Перейдите в папку вашего домена (например: public_html/academy)
3. Убедитесь, что папка assets/ существует (если нет - создайте)
4. Нажмите Upload
5. Загрузите файлы:
   - index.html → в корень домена
   - .htaccess → в корень домена
   - Все файлы из assets/ → в папку assets/
6. После загрузки проверьте права доступа:
   - Файлы: 644
   - Папки: 755

СПОСОБ 2: Через FTP (рекомендуется)
───────────────────────────────────────────────────────────────

1. Подключитесь через FTP (FileZilla, WinSCP и т.д.)
2. Перейдите в корень домена
3. Загрузите файлы:
   - index.html → корень
   - .htaccess → корень
   - Все файлы из assets/ → в папку assets/
4. Установите права доступа:
   - Файлы: 644
   - Папки: 755
