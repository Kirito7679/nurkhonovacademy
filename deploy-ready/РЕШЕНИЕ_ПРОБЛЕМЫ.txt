╚═══════════════════════════════════════════════════════════════╝

1. Откройте DevTools (F12)
2. Правый клик на кнопке обновления
3. Выберите "Очистить кэш и жесткая перезагрузка"

Или:
1. Chrome: Settings → Privacy → Clear browsing data
2. Выберите "Cached images and files"
3. Очистите

╔═══════════════════════════════════════════════════════════════╗
║                    ШАГ 4: ПРОВЕРЬТЕ В БРАУЗЕРЕ                  ║
╚═══════════════════════════════════════════════════════════════╝

1. Откройте DevTools (F12) → Network
2. Обновите страницу
3. Проверьте, какие файлы загружаются:
   - ✅ Должен: index-COOPZO17.js
   - ❌ НЕ должен: index-FK4ubfmL.js (старый)

4. Проверьте Console на ошибки:
   - Если ошибки API → проблема с Railway
   - Если "t is not defined" → старые файлы
   - Если 404 для файлов → файлы не загружены
