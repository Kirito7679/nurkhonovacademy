📋 ЧТО НУЖНО СДЕЛАТЬ:
───────────────────────────────────────────────────────────────

1. УДАЛИТЬ старые файлы из папки assets/ на сервере
   ⚠️ ВАЖНО: Удалите ВСЕ файлы из assets/ перед загрузкой новых!

2. ЗАГРУЗИТЬ новые файлы:
   - index.html → в корень домена
   - Все файлы из assets/ → в папку assets/ на сервере
   - .htaccess → в корень домена (если нужно обновить)

3. ПРОВЕРИТЬ права доступа:
   - Файлы: 644
   - Папки: 755

4. ОЧИСТИТЬ кеш браузера (Ctrl+Shift+R)

═══════════════════════════════════════════════════════════════

🔧 КАК ЗАГРУЗИТЬ:
───────────────────────────────────────────────────────────────

ВАРИАНТ 1: Через cPanel File Manager
───────────────────────────────────────────────────────────────
1. Войдите в cPanel → File Manager
2. Перейдите в папку вашего домена
3. УДАЛИТЕ все файлы из папки assets/
4. Нажмите Upload
5. Загрузите index.html в корень
6. Загрузите все файлы из assets/ в папку assets/
7. Загрузите .htaccess в корень

ВАРИАНТ 2: Через FTP (рекомендуется)
───────────────────────────────────────────────────────────────
1. Подключитесь через FTP (FileZilla, WinSCP)
2. УДАЛИТЕ все файлы из папки assets/ на сервере
3. Загрузите все файлы из этой папки (deploy-ready/)
   - index.html → корень
   - assets/* → в папку assets/
   - .htaccess → корень

═══════════════════════════════════════════════════════════════

✅ ПРОВЕРКА:
───────────────────────────────────────────────────────────────

1. Откройте сайт в браузере
2. Нажмите F12 → Console
3. Проверьте, нет ли ошибок
4. Проверьте уведомления:
   - Откройте в двух браузерах (учитель и студент)
   - Учитель одобряет доступ
   - Студент должен мгновенно получить уведомление 🔔
