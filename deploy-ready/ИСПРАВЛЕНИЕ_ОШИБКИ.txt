🔍 ПРИЧИНА:
При code splitting библиотеки, которые используют React (react-query, react-hook-form, 
react-i18next, react-player, zustand), были помещены в отдельные чанки и загружались 
раньше, чем React. Это приводило к тому, что они пытались использовать React, который 
еще не был загружен.

✅ РЕШЕНИЕ:
Все библиотеки, которые зависят от React, теперь помещены в vendor-react чанк вместе 
с React. Это гарантирует, что React загружается первым и доступен для всех зависимостей.

📦 ИЗМЕНЕНИЯ:
- react-query → vendor-react (было: vendor-state)
- react-hook-form → vendor-react (было: vendor-forms)
- react-i18next → vendor-react (было: vendor-i18n)
- react-player → vendor-react (было: vendor-player)
- zustand → vendor-react (было: vendor-state)
