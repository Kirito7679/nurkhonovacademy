📋 ЧТО НУЖНО СДЕЛАТЬ:
───────────────────────────────────────────────────────────────
Заменить 2 файла через cPanel File Manager

📁 ФАЙЛЫ ДЛЯ ЗАМЕНЫ:
───────────────────────────────────────────────────────────────
1. frontend/src/hooks/useSocket.ts
   → Заменить содержимым из: cpanel-update/useSocket.ts

2. frontend/src/components/NotificationBell.tsx
   → Заменить содержимым из: cpanel-update/NotificationBell.tsx

🔧 КАК СДЕЛАТЬ:
───────────────────────────────────────────────────────────────

ВАРИАНТ 1: Через cPanel File Manager (самый простой)
───────────────────────────────────────────────────────────────
1. Войдите в cPanel
2. Откройте "File Manager"
3. Найдите файл: frontend/src/hooks/useSocket.ts
4. Нажмите "Edit" (редактировать)
5. УДАЛИТЕ весь код в файле
6. ОТКРОЙТЕ файл cpanel-update/useSocket.ts (из этого проекта)
7. СКОПИРУЙТЕ весь код
8. ВСТАВЬТЕ в редактор cPanel
9. Нажмите "Save Changes"
10. Повторите для второго файла:
    - frontend/src/components/NotificationBell.tsx
    - Замените кодом из cpanel-update/NotificationBell.tsx

ВАРИАНТ 2: Через FTP
───────────────────────────────────────────────────────────────
1. Подключитесь через FTP (FileZilla, WinSCP и т.д.)
2. Скачайте файлы из папки cpanel-update/:
   - useSocket.ts
   - NotificationBell.tsx
3. Загрузите их, заменив существующие файлы:
   - useSocket.ts → frontend/src/hooks/useSocket.ts
   - NotificationBell.tsx → frontend/src/components/NotificationBell.tsx

✅ ПРОВЕРКА:
───────────────────────────────────────────────────────────────
1. Очистите кеш браузера (Ctrl+Shift+R)
2. Откройте платформу в двух браузерах:
   - Браузер 1: Учитель
   - Браузер 2: Студент
3. Учитель одобряет доступ студента к курсу
4. Студент должен МГНОВЕННО получить уведомление 🔔

⚠️ ВАЖНО:
───────────────────────────────────────────────────────────────
- Не забудьте сохранить изменения в обоих файлах
- После обновления перезагрузите страницу
- Если не работает, проверьте консоль браузера (F12)
